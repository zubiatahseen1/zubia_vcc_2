# Cloud Garden Interiors 🌿

**Website Project for a Biophilic Design Firm**

---

## Project Overview

**Cloud Garden Interiors** is a website for a design firm dedicated to bringing the outdoors in through the art of **biophilic design**. The firm focuses on creating spaces that connect people with nature, improving well-being, aesthetics, and functionality.

This website serves as a digital presence for the firm, showcasing:

- The philosophy and principles of biophilic design
- Portfolio of completed projects
- Services offered for residential and commercial spaces
- Contact information for consultations

The goal of the site is to reflect the beauty, calm, and sophistication of nature-inspired interior design while providing a clean, responsive, and user-friendly experience.

---

## Features

- Responsive landing page with hero section
- Navigation bar with links to multiple sections/pages
- Portfolio pages highlighting design projects
- About page detailing the firm's vision and philosophy
- Contact page with form and information for client inquiries
- Clean, minimal, and cohesive design
- Subtle use of color and imagery to evoke nature and tranquility

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git & GitHub
- Figma (for UI/UX design and mockups)
- Adobe Photoshop & Illustrator (for graphics and visuals)

---

## Design Philosophy

The website embraces **clean and minimal design principles** while emphasizing nature-inspired aesthetics. It prioritizes:

- Simplicity without sacrificing effectiveness
- Cohesive layouts that enhance user experience
- Thoughtful use of color, typography, and spacing
- Accessibility and responsiveness across all devices
